## Descripción del Proyecto
Compawnion es una aplicación para conectar a la gente con miles de perritos y gatitos en búsqueda de casa.

## Carpetas

- ### Backend
      Contiene todos los archivos del backend

  - ### controllers
        Es donde se crea la funcionalidad de los endpoints.

  - ### models
        Es donde están todos los modelos de las tablas para el proyecto.

  - ### node_modules
        Contiene todos los paquetes que se están usando.

  - ### routes
        Contiene los endpoints de cada tabla.
